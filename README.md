Features:
 - Renames Files
 - Sort all folders and files
 - Supports MP4 and MKV

How it sorts:
- you make a folder put all mp4 and mkv file in there
  sort in in the app with the season + - button
  you can drag them so it in order :)
  RECOMMENDED: if you download or get them some how mark the with like s0e1
               the app sorts them like this:
         
               Season 00      
                  Name (Year) - S00E00
                  Name (Year) - S00E01
                  Name (Year) - S00E02
               Season 01      
                  Name (Year) - S01E00
                  Name (Year) - S01E01
                  Name (Year) - S01E02
